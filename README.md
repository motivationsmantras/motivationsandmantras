# Motivations and Mantras

This project is a collection of motivational quotes and mantras to inspire focus, positivity, and discipline.

# What this project does
- Displays motivational messages
- Stores a collection of mantras
- Helps encourage daily motivation

# How to use
1. Open the project files
2. Read or display the quotes/mantras
3. Add new ones as needed

# Future ideas
